# RFMP-Secure-File-Management-Protocol
Secure Remote File Management Protocol (RFMP) using Python sockets, RSA/AES encryption, multithreading, and remote file operations.
A secure Remote File Management Protocol (RFMP) developed using Python and C as part of a Programming for Information Security course project.

## Features

* Client-server architecture using TCP sockets
* RSA public-key cryptography for secure session key exchange
* AES encryption support
* Caesar cipher encryption support
* Multithreaded server implementation
* Remote file and folder management
* File read and write operations
* Command execution on remote systems
* C client integration

## Technologies Used

* Python
* Socket Programming
* RSA Cryptography
* AES Encryption
* Multithreading
* C Programming
